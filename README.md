# BirthdayBash
A final effort for your
friend's birthday!

# How to Use:
1. Click 1 for 30 sec, 
          2 for 60 sec, 
          3 for 90 sec, 
          4 for 120 sec.

2. Exit by clicking the close window option.
3. Use arrow keys to get your present!
4. Depending on the number presents you have,
  you'll get a gift!
